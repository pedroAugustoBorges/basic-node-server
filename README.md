# webcrapping + node server

### Certificar se possui o node.js instalado
``` powershel
  node -v
  v22.20.0
```


### Criação do package.json com variaveis de ambiente
``` powershel
  npm init-y
```

### Instalação dependências node
``` powershel
 npm install express node-fetch@2 
```


### Rodando o server
``` powershel
node script.js
Proxy rodando na porta 3000
```
